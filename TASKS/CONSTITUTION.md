## ❌ АБСОЛЮТНЫЕ ЗАПРЕТЫ

### 1. ЗАПРЕЩЕНО генерировать Rust → JavaScript

Репозиторий **НЕ** является Rust проектом. Не использовать:
- `wasm-pack`
- `wasm-bindgen`
- Компиляцию `.rs` → `.js` / `.wasm`
- Любые Rust инструменты в этом репо

### 2. ЗАПРЕЩЕНО смешивать языки не по назначению

Каждый слой имеет свой язык — менять нельзя без явного приказа Генерала.

1. **Chrome Extension = только JS** — никакого TypeScript, никакого build step
2. **Node.js серверы = только TypeScript** — никакого plain JS в `src/`
3. **Не добавлять зависимости** без явного приказа Генерала
4. **Не переименовывать файлы** без явного приказа
5. **Коммитить только то что просили** — не трогать лишние файлы
6. **После каждой задачи** — отчёт в комментарии или PR
